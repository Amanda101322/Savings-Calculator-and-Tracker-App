 Digital PiggyBank Calculator & Tracker
'Save now and save your life from future misery'
A web app built for users to track savings goals with real-time logging, custom currencies, and playful Nigerian slang motivation.
Note: This app is a digital ledger—it tracks your visual progress, not real money.
 How to Use
1. Save Real Cash: Put physical money into your real bank or piggy bank.
2. Set a Target: Enter your goal (e.g., "Laptop"), choose a currency (₦, $, £, CA$, €), set an amount, and click 'Set Goal'.
3. Log Deposits: Enter the amount saved, pick a date, add an optional note, and click Deposit.
4. Track Progress: Watch the animated progress bar update and check your transaction ledger.
5. Manage App: Toggle light/dark themes or reset data anytime.

Features and Benefits
Persistent Storage: Saves goals, history, and preferences using browser `localStorage`.
Great UI: Features tactile, springy buttons and animated progress bars.
Slang Coaching: Provides real-time feedback in Pidgin English with English translations (e.g., "Omor, you never start o" to "You don land").
Zero Server Overhead: 99.9 fast performance.

 Tech Stack
HTML5: Semantic structure and forms.
CSS3: Custom variables, styling, cool palettes and keyframe animations.
JavaScript: State management, mathematical calculations, and `localStorage` integration.
